# 🔗 Linked List Mastery

A complete collection of Linked List problems covering all important patterns and variations asked in coding interviews.

This repository is part of my Data Structures & Algorithms preparation (targeting top tech companies 🚀).

---

## 📚 Topics Covered

### ✅ Basics
- Singly Linked List
- Doubly Linked List
- Circular Linked List

### 🔁 Traversal & Basics Operations
- Insertion (Head, Tail, Position)
- Deletion (Head, Tail, Position)
- Search in Linked List

### 🔄 Reversal Patterns
- Reverse a Linked List
- Reverse in K-Groups
- Reverse Between Positions

### 🔗 Fast & Slow Pointer
- Detect Cycle (Floyd’s Algorithm)
- Find Middle of Linked List
- Remove Cycle

### 🔀 Merging & Sorting
- Merge Two Sorted Lists
- Merge K Sorted Lists
- Sort Linked List (Merge Sort)

### 🧠 Advanced Problems
- Intersection of Two Linked Lists
- Palindrome Linked List
- Flatten Linked List
- Clone Linked List (with Random Pointer)

### 🔥 Hard Patterns
- LRU Cache (Using Linked List + HashMap)
- Rotate Linked List
- Partition List
